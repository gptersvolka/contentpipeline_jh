# 📘 InsightFlow Content Tracker (v0.1)

## 1) 시작하기 전에
이 문서는 마케팅 콘텐츠의 배포 및 성과 추적을 위한 **"UTM 링크 생성 및 관리 시스템"** 구축을 목표로 한다.
동료가 생성한 콘텐츠 URL에 캠페인 식별자를 결합하여, 플랫폼별(인스타, 스레드, 유튜브, 블로그) 추적 링크를 원클릭으로 생성한다.

## 2) 예상 효과
* **표준화:** 수동으로 입력하던 UTM 파라미터 오타 방지 및 규칙 통일
* **속도 향상:** URL 복사 → 엑셀 입력 → 변환 → 복사 과정을 1초로 단축
* **데이터 정합성:** Mixpanel 등 데이터 분석 툴에서 일관된 Campaign Name으로 성과 조회 가능

## 3) 완주 플랜 (Roadmap)
이번 v0.1 버전은 **"링크 생성 및 복사 UI"** 완성을 목표로 한다.

* **🚀 Phase 1 - UI 및 로직 구현**
    * Target URL 및 Campaign Name 입력 폼
    * 4개 채널(Instagram, Threads, YouTube, Blog) 자동 변환 로직
    * 클립보드 복사 기능
* **🚀 Phase 2 - 배포 및 공유**
    * GitHub 리포지토리 생성
    * Vercel을 통한 웹 배포 (팀원 접속용)

## 4) 시스템 구조
[User UI]
   ↓ (Original URL, Campaign Name 입력)
[Generator Logic]
   ├── URL 유효성 검사
   ├── 채널별 UTM 파라미터 결합 (표준 규칙 적용)
   ├── Short Link 변환 (추후 도입, v0.1은 제외)
   ↓
[Output View]
   ├── 4개 채널별 완성된 링크 카드 표시
   └── [Copy] 버튼 제공

## 5) 데이터 스키마 (UTM 규칙)
모든 링크는 공통적으로 입력받은 `Original URL` 뒤에 파라미터를 붙인다.

| 채널 (Channel) | utm_source | utm_medium | utm_campaign | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **Instagram** | `instagram` | `social` | *(사용자 입력값)* | 프로필 링크용 |
| **Threads** | `threads` | `social` | *(사용자 입력값)* | 스레드 본문용 |
| **YouTube** | `youtube` | `video` | *(사용자 입력값)* | 설명란용 |
| **Blog** | `blog` | `seo` | *(사용자 입력값)* | 콘텐츠 내 링크 |

*예시:*
* 입력: `https://gpters.org/study` + 캠페인명: `2024_spring_event`
* 결과(인스타): `https://gpters.org/study?utm_source=instagram&utm_medium=social&utm_campaign=2024_spring_event`

## 6) 구현 상세 요구사항
### 6-1. UI/UX
* **톤앤매너:** InsightFlow 브랜드 컬러(Blue #007AFF)와 화이트 배경, 깔끔한 카드 UI
* **Input:**
    * `Target URL`: 필수 입력
    * `Campaign Name`: 필수 입력 (영문/숫자/하이픈만 허용하도록 가이드)
* **Action:** [Generate Links] 버튼 클릭 시 하단 결과 표시
* **Output:** 채널 아이콘, 채널명, 생성된 URL, [Copy] 버튼이 있는 리스트 형태

### 6-2. 기술 스택
* **Frontend:** React + Vite (빠른 개발 및 빌드)
* **Styling:** Tailwind CSS (빠르고 직관적인 스타일링)
* **Deployment:** Vercel (GitHub 연동 자동 배포)

## 7) 향후 확장 (v0.2)
